How to assign a layout to this device:

  1. Open the Digital Signage Management Server in your browser
  2. Navigate to Device Management tab
  3. Find this device using its Client ID or IP Address
  4. Select a layout from the Assigned Layout dropdown
  5. The device will automatically receive and display the new layout

Troubleshooting:

  1. Device not showing in server? Check network connection and verify server URL is correct
  2. Layout not displaying? Use "Restart Client" button above to refresh the display
  3. Performance issues? Use "Clear Cache" to free up disk space
  4. Connection problems? Check that the server is running and firewall allows connections
